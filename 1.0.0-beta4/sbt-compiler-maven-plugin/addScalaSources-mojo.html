sbt-compiler:addScalaSources

Full name:

com.google.code.sbt-compiler-maven-plugin:sbt-compiler-maven-plugin:1.0.0-beta4:addScalaSources

Description:

Add default Scala source roots, if they exist and are not already added.

Helper mojo.
Adds:
  • src/main/scala as compile source root
  • src/test/scala as test compile source root

Attributes:

  • Requires a Maven project to be executed.
  • Since version: 1.0.0.
  • Binds by default to the lifecycle phase: initialize.

Parameters

(no parameters)
